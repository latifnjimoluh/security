# Travaux Pratiques en Cybersécurité

## Description
Une collection de travaux pratiques couvrant divers aspects de la cybersécurité, incluant l'analyse de réseaux, la sécurité applicative et les tests d'intrusion.

## Structure du Projet
- `tp1/` : Premier travail pratique.
- `tp2/` : Deuxième travail pratique.
- `tp3_orange/` : Travail pratique focalisé sur l'analyse de sécurité pour un opérateur (Orange).

## Stack Technique
- Outils d'analyse de sécurité.
- Scripts Python / Shell (selon le TP).
- Rapports au format PDF/Markdown.
